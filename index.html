Steps to follow for contributing to this Sandbox Project

[Click on the steps given below to learn, how to perform them]
  1. Install Git on your PC
  2. Fork and Commit
    • Fork this repository
    • Clone your forked repo on your own PC
    • Change anything you want like image, bgcolor, text, font, etc. (except this list)
    • Push changes to your repo
    • Create a pull request to main repo
  3. That's It! You've contributed for an open source project now
    Keep Coding and changing some stuff around here till you become comfortable with git commands
  4. Ask any queries in the Discord Server so that everyone can help eachother in solving problems we face

  5. You may also contact using any of the following ways :

      Email to - devscollab@gmail.com
